## 🚀 Usage

1. Prerequisits
    - Windows 10  (Build 10.0.17763.0) onwards and Windws 11
    - Visual Studio 2022 with `slnx` enabled
    - .NET 9 SDK
2. Clone the repo
    ```console
    git clone https://github.com/0x5bfa/JumpListViewer.git
    ```
3. Open `JumpListViewer.slnx`
4. Build the solution
